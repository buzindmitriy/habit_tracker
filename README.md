# Habit Tracker

Habit Tracker — это Django-приложение для отслеживания привычек пользователей. Проект использует Docker для контейнеризации и включает бэкенд, базу данных PostgreSQL, Redis, Celery и Celery Beat.

## Требования

Для запуска проекта вам потребуется:

- **Docker**: Убедитесь, что Docker установлен на вашей системе.
  - [Скачать Docker](https://www.docker.com/)
- **Docker Compose**: Убедитесь, что Docker Compose установлен (обычно он входит в состав Docker Desktop).
- **Git**: Для клонирования репозитория.
